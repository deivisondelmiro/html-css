Primeiros passos HTML

Editores

Entender que os códigos, tanto em HTML, quanto em CSS, e tantos outros, precisam e recomenda-se serem feitos em editores dedicados a essa função, que é a criação de códigos.
Os mais conhecidos, são: Visual Studio Code; Atom; Sublime; Brackets; Notepad ++.

Informações

HTML > HiperText Language Markup. É uma linguagem de marcação, em que definará por maio de tags < > o que cada uma terá e representará no documento. Contudo, no geral,o HTML irá tratar do conteúdo. Por meio dos hiperlinks por assim dizer; seria o esqueleto.
CSS > Cascading Style Sheets. São as folhas de estilos, responsáveis pela estilização, o visual de um documento em HTML. Sendo a parte de cima de um esqueleto, talvez, a pele, e até mesmo as roupas.
JS > JavaScript. Uma linguagem de programação, mais para a web, e que seria como se fosse os movimentos, ou até mesmo os órgãos. Ou seja, é o que dá as funções aos elemntos que estão dentro do HTML. Faz a interação.
